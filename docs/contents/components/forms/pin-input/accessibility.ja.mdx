## キーボード操作

| キー         | 説明                                         | 状態 |
| ------------ | -------------------------------------------- | ---- |
| `Tab`        | フォーカスを次の入力フィールドに移動します。 | -    |
| `ArrowRight` | フォーカスを次の入力フィールドに移動します。 | -    |
| `ArrowLeft`  | フォーカスを前の入力フィールドに移動します。 | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性   | 使い方                         |
| -------------- | -------------- | ------------------------------ |
| `PinInput`     | `role="group"` | グループであることを示します。 |

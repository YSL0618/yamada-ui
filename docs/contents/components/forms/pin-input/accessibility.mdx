## Keyboard Navigation

| Key          | Description                                 | State |
| ------------ | ------------------------------------------- | ----- |
| `Tab`        | Move the focus to the next input field.     | -     |
| `ArrowRight` | Move the focus to the next input field.     | -     |
| `ArrowLeft`  | Move the focus to the previous input field. | -     |

## ARIA Roles and Attributes

| Component  | Role and Attributes | Usage                           |
| ---------- | ------------------- | ------------------------------- |
| `PinInput` | `role="group"`      | Indicates that this is a group. |
